# Public API

The 555x402 API allows developers to integrate the Streaming OS into their own applications.

**Base URL**: `https://api.rendernet.work`

## Authentication

All requests to `/pub/v1` require an API Key.
`X-API-Key: <your_api_key>`

## Endpoints

### PayLinks (Hyperlink)

#### Create PayLink
`POST /pub/v1/links`
Creates a new Hyperlink for payments or engagement.

```json
{
  "creatorId": "creator_123",
  "chainType": "solana",
  "model": "engagement",
  "splits": {
    "arp": 1000,
    "creator": 4500,
    "platform": 4500
  }
}
```

> **Note**: Splits are in basis points of gross revenue. ARP (10%) is taken first, then the remaining 90% is split 50/50 between creator and platform.

#### Get PayLink
`GET /pub/v1/links/{code}`
Retrieves metadata for a specific link.

### Payments (AGG)

#### Verify Payment
`POST /pub/v1/payments/verify`
Verifies a 402 payment header and returns a settlement proof.

#### Payment Status
`GET /pub/v1/payments/{id}`
Checks the status of a transaction.

### Attention (VAP)

#### Request Token
`POST /pub/v1/attention/token`
Requests a session token for tracking engagement.

### Points & Credits

#### Get Balance
`GET /pub/v1/points/balance?wallet={address}`
Returns the points and credits balance for a wallet.

#### Get Leaderboard
`GET /pub/v1/leaderboard?game={appId}&period={day|week|month|year}`
Returns leaderboard rankings for a game and time period.

### Cross-Chain Settlement (sw4p)

> **Note**: For full cross-chain bridge functionality, see the [sw4p API documentation](https://docs.sw4p.io).

#### Get Transfer Status
`GET /pub/v1/bridge/status/{id}`
Checks the status of a cross-chain settlement.

#### Estimate Fees
`GET /pub/v1/fees/estimate`
Estimates the gas and platform fees for a cross-chain transfer.

```json
{
  "from": "BASE",
  "to": "SOLANA",
  "amount": "100.00"
}
```

**Response:**
```json
{
  "totalFee": "1.50 USDC",
  "platformFee": "0.50 USDC",
  "gasFee": "1.00 USDC",
  "estimatedTime": "60 seconds"
}
```

### Economy

#### ARP Summary
`GET /api/internal/economics/arp-summary`
Returns the current ARP distribution summary (internal).

#### Settlement Status
`GET /api/internal/economics/settlement-status`
Returns the status of the weekly settlement engine (internal).
