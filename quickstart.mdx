## Get started in three steps

Transform your streaming career with blockchain-powered tools and sustainable monetization.

### Step 1: Connect Your Wallet

<AccordionGroup>
  <Accordion icon="wallet" title="Set up crypto wallet">
    Connect your preferred crypto wallet to start earning RNDR tokens and access premium streaming features.

    **Supported Wallets:**
    - Phantom (Solana)
    - MetaMask (Ethereum)
    - Solflare
    - Trust Wallet

    <Tip>RNDR tokens power all gamification and monetization features.</Tip>

  </Accordion>
  <Accordion icon="gamepad" title="Set up 555 Browser Source">
    Add the 555 gamification overlay to your stream using OBS browser source.

    1. In OBS, add a "Browser Source"
    2. Set URL to: https://555.rendernet.work/overlay
    3. Set dimensions to 1920x1080
    4. Enable "Shutdown source when not visible"
    5. Connect your wallet when prompted

    <Tip>Your first prediction market launches automatically after setup.</Tip>

  </Accordion>
</AccordionGroup>

### Step 2: Configure Your Stream

<AccordionGroup>
  <Accordion icon="settings" title="Set up OBS/Streamlabs">
    Integrate Render Network tools with your streaming software for seamless gamification.

    **OBS Studio Setup:**
    1. Add Browser Source for 555 overlay
    2. Configure wallet connection
    3. Set up prediction market displays

    <Tip>Use the 555 extension for automatic overlay positioning.</Tip>

</Accordion>
<Accordion icon="gamepad" title="Create Your First Game">
    Launch your first interactive stream with blockchain rewards.

    1. Choose game type (Prediction, Trivia, Mini-games)
    2. Set RNDR token entry fees
    3. Configure prize distribution
    4. Go live and watch engagement soar

    <Tip>Start with small bets to build community trust.</Tip>

  </Accordion>
</AccordionGroup>

### Step 3: Start Earning

<Accordion icon="dollar-sign" title="Launch Monetization">
  Transform your stream into a sustainable income source with blockchain-powered tools.

1. Enable tip automation
2. Set up RNDR token rewards
3. Configure fee distribution
4. Track earnings in real-time
   </Accordion>
