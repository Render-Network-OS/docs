Integrate Render Network's powerful tools into your streaming and development workflow.

<Steps>
<Step title="Set up Your Wallet">

Connect your crypto wallet to start earning RNDR tokens:

```javascript
// Wallet connection example
import { useWallet } from "@solana/wallet-adapter-react";

const { connect, publicKey } = useWallet();

// Connect to start earning
await connect();
```

**Supported Wallets:**

- Phantom (Recommended)
- Solflare
- MetaMask
- Trust Wallet

</Step>

<Step title="Set up 555 Browser Source">

Add the 555 gamification overlay using OBS browser source:

```bash
# Add Browser Source in OBS:
# 1. Right-click Sources → Add → Browser Source
# 2. Name: "555 Overlay"
# 3. URL: https://555.rendernet.work/overlay
# 4. Width: 1920, Height: 1080
# 5. Enable "Shutdown source when not visible"
# 6. Click OK and connect wallet
```

**Browser Source Features:**

- Real-time prediction markets
- Interactive mini-games
- Automated prize distribution
- Seamless stream integration

</Step>
</Steps>

## API Integration

Connect your applications with Render Network APIs:

**Authentication:**

```javascript
const response = await fetch("/api/rndr/balance", {
  headers: {
    Authorization: `Bearer ${wallet.signature}`,
    "Content-Type": "application/json"
  }
});
```

**Real-time Updates:**

```javascript
// WebSocket connection for live game updates
const ws = new WebSocket("wss://api.rendernet.work/games");

ws.onmessage = (event) => {
  const gameUpdate = JSON.parse(event.data);
  updateGameState(gameUpdate);
};
```

## Revenue Tracking

Monitor your earnings in real-time with comprehensive analytics:

**Dashboard Access:**

```javascript
// Get earnings summary
const earnings = await fetch("/api/earnings/summary", {
  headers: { Authorization: `Bearer ${wallet.signature}` }
});

const data = await earnings.json();
console.log(`Total Earned: ${data.totalRNDR} RNDR`);
```

**Real-time Updates:**

- Live viewer engagement metrics

- Game performance analytics
- Community growth statistics

## SDK Installation

Install Render Network SDK for advanced integrations:

```bash
npm install @render-network/sdk

# Or with yarn
yarn add @render-network/sdk
```

**Basic Usage:**

```javascript
import { RenderNetwork } from "@render-network/sdk";

const rn = new RenderNetwork({
  wallet: yourWallet,
  network: "mainnet-beta"
});

// Create a prediction market
const market = await rn.createPredictionMarket({
  question: "Will I win?",
  options: ["Yes", "No"],
  duration: 300
});
```

## Troubleshooting

<AccordionGroup>
  <Accordion title='Wallet Connection Failed'>

    **Common Solutions:**
    1. Ensure your wallet is unlocked and connected to Solana
    2. Check if you have sufficient SOL for transaction fees
    3. Verify network settings (mainnet-beta)
    4. Clear browser cache and reconnect

    **Need Help?** Join our [Discord](https://discord.gg/render-network)

  </Accordion>

  <Accordion title="Games Not Loading">

    **Debugging Steps:**
    1. Check 555 extension is properly installed
    2. Verify OBS/browser permissions
    3. Clear extension cache
    4. Restart streaming software
    5. Check network connectivity

    **Support:** Visit [rendernet.work/support](https://rendernet.work/support)

  </Accordion>

  <Accordion title="RNDR Token Issues">

    **Token Problems:**
    1. Verify wallet supports SPL tokens
    2. Check token balance on Solana explorer
    3. Ensure correct token contract address
    4. Wait for transaction confirmation

  </Accordion>
</AccordionGroup>
