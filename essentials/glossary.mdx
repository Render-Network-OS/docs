---
title: "Glossary"
description: "Key terms across the Render Network ecosystem"
---

- **VRF:** Verifiable Random Function; cryptographic randomness with proofs
- **Streamflow:** Token locking/vesting protocol used for $555 locks
- **SPL:** Solana Program Library token standard
- **PDA:** Program Derived Address used by Anchor programs
- **Anchor:** Framework for Solana program development
- **Pump.fun:** Solana-native token launch and trading environment
- **Switchboard/Pyth:** Oracle providers; Switchboard for VRF
- **AMM:** Automated Market Maker used for prediction markets and swaps
- **stream_id:** Identifier tagging entries per chat/stream
- **Buybacks:** Market purchases of tokens using collected fees
- **Rollover:** Carry-over of unclaimed or pooled amounts to future rounds
