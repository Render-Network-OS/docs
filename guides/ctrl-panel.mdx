## Overview

Use CTRL to manage visuals and interactive controls without external software. Connect your Solana wallet to unlock creator tools and granular permissions.

## Onboarding

<Steps>
  <Step title="Connect Wallet">
    Connect your Solana wallet (Phantom/Solflare). The owner wallet defines
    stream control and module access.
  </Step>
  <Step title="Select Stream">
    Choose the target stream and load an existing configuration or start from a
    template.
  </Step>
  <Step title="Publish & Verify">
    Publish overlays and verify they render correctly (native or OBS Browser
    Source fallback).
  </Step>
</Steps>

## Themes & Overlays

- Create and edit themes with real-time preview
- Manage overlay packs (ticker, alerts, chat, goals)
- Export/import configurations for reuse across streams

## Alerts & Badges

- Configure triggers for follows, tips, achievements
- Assign rarity tiers and seasonal badges
- Tie triggers to on-chain events where applicable

## Token Connect & Gating

- Gate specific actions (overlay triggers, chat modes) behind $555 thresholds
- Per-stream token verification rules with minimum holdings
- Delegate moderator wallets with scoped permissions

## Troubleshooting

- If overlays don’t render, republish and reload the scene
- Check wallet connection status and permissions
- Verify minimum $555 holdings for gated actions
