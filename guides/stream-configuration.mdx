## Game Configuration

Set up and customize your Render Network games for maximum engagement and revenue optimization.

## Prediction Markets

### Basic Setup

<Steps>
  <Step title="Create Prediction Market">
    Set up your first prediction market:

    ```json
    {
      "marketType": "prediction",
      "question": "Will I win this match?",
      "options": ["Yes", "No"],
      "duration": 300,
      "minBet": "1 RNDR",
      "maxBet": "100 RNDR",
      "houseEdge": 0.02
    }
    ```

  </Step>

  <Step title="Configure Payouts">
    Set up automatic payout distribution:

    ```json
    {
      "payoutStructure": {
        "streamerShare": 0.7,
        "winnerPool": 0.3,
        "platformFee": 0.02
      },
      "autoPayout": true,
      "instantResolution": false
    }
    ```

  </Step>

  <Step title="Set Visual Theme">
    Customize the market appearance:

    ```json
    {
      "theme": {
        "colors": {
          "primary": "#FF6B35",
          "secondary": "#0b8cb7",
          "background": "rgba(0,0,0,0.8)"
        },
        "fonts": {
          "primary": "Inter",
          "size": "16px"
        },
        "animations": {
          "duration": 300,
          "easing": "ease-out"
        }
      }
    }
    ```

  </Step>

  <Step title="Test & Launch">
    Preview and launch your prediction market:

    ```javascript
    // Test market
    const testResult = await market.testConfiguration();

    // Launch live
    if (testResult.valid) {
      await market.launch();
      console.log("Prediction market is live!");
    }
    ```

  </Step>
</Steps>
