## Installation & Setup

<Steps>
  <Step title="Download & Install">
    Choose your preferred wallet and install it:

    **Browser Extension:**
    ```bash
    # Chrome Web Store
    Search for "[Wallet Name]"

    # Firefox Add-ons
    Search for "[Wallet Name]"

    # Edge Add-ons
    Search for "[Wallet Name]"
    ```

  </Step>

  <Step title="Create New Wallet">
    Set up your wallet securely:

    **Security Checklist:**
    - ✅ Create strong, unique password
    - ✅ Write down recovery phrase (24 words)
    - ✅ Store recovery phrase offline and secure
    - ✅ Enable 2FA if available
    - ✅ Never share private keys or seed phrases

  </Step>

  <Step title="Fund Your Wallet">
    Add cryptocurrency to start using Render Network:

    **Funding Options:**
    - Buy crypto directly in wallet
    - Transfer from exchange
    - Use on-ramp services
    - Receive from other wallets

    **Minimum Balance:** 0.01 SOL for transaction fees

  </Step>

  <Step title="Connect to Render Network">
    Link your wallet to start earning:

    ```javascript
    // Connect to Render Network
    const renderNetwork = new RenderNetwork();
    await renderNetwork.connectWallet(wallet);

    // Verify connection
    const balance = await renderNetwork.getRNDRBalance();
    console.log(`Connected! RNDR Balance: ${balance}`);
    ```

  </Step>
</Steps>
