Integrate Render Network's powerful gamification tools with OBS Studio for seamless streaming enhancement.

<Steps>
<Step title="Add Browser Source">

Add Render Network's 555 overlay as a browser source in OBS:

```bash
# Browser Source Configuration:
# 1. Click "+" in Sources panel → Browser Source
# 2. Name: "Render Network 555 Overlay"
# 3. URL: https://555.rendernet.work/overlay
# 4. Width: 1920, Height: 1080
# 5. Enable "Shutdown source when not visible"
# 6. Enable "Custom audio device" if needed
# 7. Click OK
```

**Browser Source Settings:**

- **URL**: `https://555.rendernet.work/overlay`
- **Width**: 1920 pixels (full HD)
- **Height**: 1080 pixels (full HD)
- **FPS**: 60 (recommended)
- **Shutdown when not visible**: Enabled

</Step>

<Step title="Connect Your Wallet">

Connect your crypto wallet to enable RNDR token earning:

```javascript
// Wallet connection in overlay
// 1. Click "Connect Wallet" in the overlay
// 2. Select your wallet (Phantom recommended)
// 3. Approve the connection
// 4. Sign the authentication message
```

**Supported Wallets:**

- **Phantom** (Recommended for streaming)
- **Solflare**
- **MetaMask**
- **Trust Wallet**

</Step>
</Steps>

Enhance your streaming setup with advanced OBS features and Render Network integration:

**Browser Source Automation:**

```javascript
// OBS WebSocket API integration
const obs = new OBSWebSocket();
await obs.connect("ws://localhost:4444", "your-password");

const browserSource = {
  resource: "BrowserSource",
  args: {
    name: "555 Overlay",
    url: "https://555.rendernet.work/overlay",
    width: 1920,
    height: 1080,
    shutdown: true
```

## SDK Installation

Install Render Network SDK for advanced integrations:

```bash
npm install @render-network/sdk

# Or with yarn
yarn add @render-network/sdk
```

**Basic Usage:**

```javascript
import { RenderNetwork } from "@render-network/sdk";

const rn = new RenderNetwork({
  wallet: yourWallet,
  network: "mainnet-beta"
});

// Create a prediction market
const market = await rn.createPredictionMarket({
  question: "Will I win?",
  options: ["Yes", "No"],
  duration: 300
});
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Browser Source Not Loading">

    **Common Solutions:**
    1. Verify the URL is correct: `https://555.rendernet.work/overlay`
    2. Check OBS browser source settings (width/height)
    3. Enable "Shutdown source when not visible"
    4. Restart OBS Studio
    5. Check firewall/antivirus blocking the connection

    **Need Help?** Join our [Discord](https://discord.gg/render-network)

  </Accordion>

  <Accordion title="Wallet Connection Issues">

    **Debugging Steps:**
    1. Ensure your wallet extension is installed and updated
    2. Check if wallet is unlocked and connected to Solana
    3. Verify you have sufficient SOL for transaction fees
    4. Clear browser cache and reload the browser source
    5. Try different wallet (Phantom recommended)

    **Support:** Visit [rendernet.work/support](https://rendernet.work/support)

  </Accordion>

  <Accordion title="OBS Performance Issues">

    **Performance Optimization:**
    1. Lower FPS setting in browser source (try 30fps)
    2. Reduce browser source resolution if needed
    3. Enable "Shutdown source when not visible"
    4. Close unnecessary browser tabs
    5. Update graphics drivers
    6. Check CPU/GPU usage during streaming

  </Accordion>

  <Accordion title="Audio/Video Sync Problems">

    **Sync Solutions:**
    1. Enable "Custom audio device" in browser source
    2. Check OBS audio settings
    3. Adjust browser source FPS to match stream
    4. Use hardware acceleration if available
    5. Test with different browsers

  </Accordion>
</AccordionGroup>
