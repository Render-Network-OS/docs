## Persistence

- Use GORM for all database interactions; avoid raw SQL
- Define clear models and migrations; enforce constraints
- Wrap multi-step writes in transactions

## Access Control & Limits

- RBAC: owner, moderator, viewer roles
- Rate limits per user/stream; burst controls
- Audit trails for sensitive actions

## Secrets & Config

- Server-only environment variables
- No secrets in frontend bundles
- Rotations and incident playbooks
