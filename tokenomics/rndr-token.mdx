### Code Examples

<AccordionGroup>
  <Accordion title="JavaScript Integration" icon="code">
    **Basic RNDR Transfer:**
    ```javascript
    import { Connection, PublicKey, Transaction } from '@solana/web3.js';
    import { TOKEN_PROGRAM_ID, getAssociatedTokenAddress, createTransferInstruction } from '@solana/spl-token';

    const connection = new Connection('https://api.mainnet.solana.com');
    const sender = new PublicKey('YOUR_WALLET_ADDRESS');
    const recipient = new PublicKey('RECIPIENT_ADDRESS');
    const rnrdMint = new PublicKey('RNDR_CONTRACT_ADDRESS');

    // Get associated token accounts
    const senderATA = await getAssociatedTokenAddress(rnrdMint, sender);
    const recipientATA = await getAssociatedTokenAddress(rnrdMint, recipient);

    // Create transfer instruction
    const transferInstruction = createTransferInstruction(
      senderATA,
      recipientATA,
      sender,
      1000000000, // 1 RNDR (9 decimals)
      [],
      TOKEN_PROGRAM_ID
    );
    ```
  </Accordion>

  <Accordion title="Python Integration" icon="python">
    **Balance Query Example:**
    ```python
    from solana.rpc.api import Client
    from spl.token.client import Token
    from spl.token.constants import TOKEN_PROGRAM_ID

    # Connect to Solana
    client = Client("https://api.mainnet.solana.com")

    # RNDR contract address
    rnrd_mint = "RNDR_CONTRACT_ADDRESS"

    # Query balance
    token = Token(client, rnrd_mint, TOKEN_PROGRAM_ID, None)
    balance = token.get_balance("YOUR_WALLET_ADDRESS")

    print(f"RNDR Balance: {balance.value.ui_amount}")
    ```
